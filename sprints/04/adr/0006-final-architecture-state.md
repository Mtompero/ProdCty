# ADR 0006 – Végleges backend architektúra

## Döntés

A projekt backend része Node.js + egyszerű service-alapú logikával valósult meg.

## Indoklás

- Gyors fejlesztés
- Könnyű tesztelhetőség
- Vertikális szelet gyors validálása

## Következmény

A rendszer jól bővíthető frontend és további API funkciók irányába.
