## Projekt röviden

A ProdCty egy audiofókuszú közösségi webalkalmazás producerek és zenészek számára. A felhasználók royalty-free sample-öket tölthetnek fel és böngészhetnek, illetve külön demo boardon oszthatják meg a munkáikat szakmai visszajelzésre.
