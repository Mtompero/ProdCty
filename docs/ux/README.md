## Projekt röviden

A ProdCty egy audiofókuszú közösségi webalkalmazás producerek és zenészek számára. A felhasználók royalty-free sample-öket tölthetnek fel és böngészhetnek, illetve work-in-progress demókat oszthatnak meg pontszámos, szöveges visszajelzésre.

A jelenlegi frontend React + TypeScript + Vite alapú, a forrása a `frontend-react/` mappában található. A korábbi külön közösségi feedback nézet funkciói a Demos oldalba kerültek: a demók böngészése, lejátszása, upvote/downvote kezelése, collab request küldése, valamint a pontszámos, szöveges feedback kezelése egy széles popupban történik. A bejövő és elküldött együttműködési kérések a fejlécben elérhető Inbox panelen követhetők.
